# open.gismo 0.1.0

- Initial public release of the open.gismo package.
- Added GitHub-backed `lConfig` hooks for reading and writing workflow inputs, outputs, and execution status.
- Added project snapshot helpers for creating, listing, and inspecting snapshot runs.
- Added a static workflow explorer front-end with workflow metadata, package manifests, logs, and artifact browsing.
- Added example project assets under `inst/examples/demo-study` for development and testing.
- Hardened site CSV parsing and preserved the intended compact table header presentation before release.
